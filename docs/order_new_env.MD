# Process for Requesting a New Environment in AWS

## Submitting a Request for a New Environment

- Open a GitHub ticket with the following content:
    * Environment Purpose
    * Required resources: vCPU / RAM / HDD
    * Required cluster components
    * Planned usage period for the environment
    * List of people who need access

- [Example request](https://github.com/Netcracker/qubership-workflow-hub/issues/284)

---

## We will calculate the cost and sends a request for approval

***After approval***
- A new user account is created in AWS (if necessary)
- The requested environment is created
- The requested components are installed
- Access is granted
